# Anzdrop

プライバシーを重視したファイル共有サービスです。

## 概要
Anzdrop は、
「運営者すらファイルの中身を閲覧できない」ことを目指した
エンドツーエンド暗号化型のファイル共有サービスです。

主な特徴：
- クライアント側での暗号化
- 一時的な共有URLの発行
- 自動削除（一定期間後）
- プライバシー最優先設計

## 技術スタック（予定）
- フロントエンド: Next.js
- API: Hono
- インフラ: Cloudflare
- ストレージ: Cloudflare R2
- データベース: Cloudflare D1
- 暗号化: WebCrypto API
